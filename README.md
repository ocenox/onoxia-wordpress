# ONOXIA - KI-Chatbot for WordPress

KI-gestützter Chat-Widget mit RAG-Wissensbasis, Live-Chat und DSGVO-Konformität. Auto-Sync für Seiten, WooCommerce-Produkte, llms.txt und Sitemap.

## Features

- **Auto-Wissensimport** — Seiten und Beiträge werden automatisch zur Bot-Wissensbasis
- **WooCommerce-Integration** — Produktkatalog-Sync, Produkt-Kontext auf Produktseiten
- **llms.txt + Sitemap Import** — Bot kennt sofort die ganze Website
- **Kontextsensitives RAG** — Tags und URL-Filter steuern welches Wissen der Bot nutzt
- **Live-Chat** — Handover an Support-Agents
- **DSGVO by Design** — Keine Cookies, IP-Hashing, EU-KI-Anbieter
- **Shadow DOM** — Garantiert keine CSS-Konflikte mit Ihrem Theme

## Installation

1. Plugin installieren und aktivieren
2. Unter **Einstellungen → ONOXIA** den API-Token eingeben
3. Token erstellen unter [onoxia.nz/app/api-tokens](https://onoxia.nz/app/api-tokens)
