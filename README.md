# ONOXIA – AI Chatbot for WordPress

AI-powered chat widget with RAG knowledge base, live chat handover and GDPR compliance. Auto-sync for pages, WooCommerce products, llms.txt and sitemaps.

## Features

- **Auto Knowledge Import** — Pages and posts are automatically synced to the bot knowledge base
- **WooCommerce Integration** — Product catalogue sync, product context on product pages
- **llms.txt + Sitemap Import** — Bot instantly knows your entire website
- **Context-sensitive RAG** — Tags and URL filters control which knowledge the bot uses
- **Live Chat** — Handover to human support agents
- **GDPR by Design** — No cookies, IP hashing, EU-based AI provider
- **Shadow DOM** — Guaranteed no CSS conflicts with your theme

## Installation

1. Install and activate the plugin
2. Go to **Settings → ONOXIA** and enter your API token
3. Create a token at [onoxia.nz/app/api-tokens](https://onoxia.nz/app/api-tokens)
